### 第二题 动态规划

dp[i][j]表示s[i]和s[j]之间的字符串是否为回文串，是为1，不是为0。

其地推方程为

当s[i] == s[j], dp[i][j] = dp[i+1][j-1];

当s[i] != s[j], dp[i][j] = 0;

初始化为(即，得到子串长度为1和2的值)

1) dp[i][i] = 1;

2) if(s[i] == s[i+1]) dp[i][i+1] = 1;

递推的时候采用，子串长度不断变增加的方式进行求解。

```c++
for(int L = 3; L < len; L++) {
  for(int i = 0; i + L - 1 < len; i++) {
    int j = i + L - 1;
    if(s[i] == s[j] && dp[i+1][j-1] == 1) {
      dp[i][j] = 1;
      ans = L;
    }
  }
}
```
